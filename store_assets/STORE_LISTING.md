## 긴 설명 (최대 4000자)

```
■ FIRST ORDER: BRICK CHAOS

끝없이 내려오는 브릭의 폭포를 막아라!

【게임 소개】
First Order는 클래식 브레이크아웃 장르에 로그라이트 RPG 요소를 결합한
모바일 아케이드 게임입니다. 볼이 떨어져도 게임이 끝나지 않습니다.
패들에 부딪히면 볼이 되살아나 다시 브릭을 향해 날아갑니다.

【핵심 메카닉】
• 브릭 폭포: 위에서 끊임없이 쏟아지는 브릭 행렬
• 볼 재활용: 볼이 바닥에 떨어지면 패들 위에 자동 재배치
• 브릭이 패들에 닿으면 게임 오버 — 집중력이 생명!

【로그라이트 퍽 시스템】
레벨업마다 3개의 퍽 중 1개를 선택해 이번 런의 빌드를 완성하세요.

★ Common
  · 마그넷 바: 패들 너비 +20%
  · 오버드라이브: 볼 속도 ×1.2 (최대 3중첩)
  · 경험치 증폭: EXP 획득 +30%
  · 큰 볼: 볼 크기 및 히트박스 ×1.3
  · 빠른 소환: +볼 쿨다운 -3초

★★ Uncommon
  · 고스트 볼: 브릭 1개 관통
  · 헤비 메탈: 볼 파워 +1, 속도 소폭 감소
  · 쉴드: 볼 이탈 시 자동 반사
  · 분열 볼: 파괴 시 15% 확률로 볼 추가 생성
  · 모멘텀: 연속 타격으로 데미지 스택

★★★ Rare
  · 전격 볼: 번개로 같은 열 랜덤 브릭 추가 피해
  · 폭발탄: 파괴 시 20% 확률 3×3 폭발
  · 노바 버스트: 패들 반사마다 충격파
  · 인페르노: 파괴 시 같은 행 전체 화염 전파

★★★★ Legendary
  · 히드라 바: 패들 반사 시 볼 3분열
  · 타임 스탑: 패들 반사마다 브릭 0.8초 동결

【전략적 시너지】
퍽 조합에 따라 완전히 다른 플레이가 펼쳐집니다.
히드라 + 노바 버스트로 화면을 폭발로 뒤덮거나,
타임 스탑 + 히드라로 브릭을 사실상 정지시켜보세요.

【영구 업그레이드 시스템】
런을 거듭할수록 샤드를 모아 영구 업그레이드를 해금하세요.
시작 볼 속도, 패들 너비, 쿨다운 감소 — 매 런이 더 강해집니다.

【특징】
• 광고 없음
• 인터넷 연결 불필요
• 세로 화면 최적화 (9:16)
• 한 손 조작
```

## 스크린샷 촬영 가이드

Android Studio 에뮬레이터 (Pixel 7 Pro, 1080×2400) 기준:

1. **메인 메뉴** — 타이틀 + 최고점 + 샤드 표시
2. **게임 초반** — 브릭 3~4줄, 볼 1개, 패들 조작
3. **레벨업 퍽 선택** — PerkScene (3카드 표시)
4. **퍽 조합 활성화** — 레벨 5~8, 여러 이펙트 동시 발동
5. **런 요약** — RunSummaryScene, 퍽 목록 + 샤드 획득
6. **업그레이드 화면** — UpgradeScene

스크린샷 최소 2장, 권장 6장 (9:16 비율).
