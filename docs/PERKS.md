# 퍽 목록

레벨업 시 3개 중 1개를 선택한다. 퍽은 희귀도(Common → Uncommon → Rare → Legendary)에 따라 구분되며, 일부 퍽은 중복 선택이 가능하다.

---

## ★ Common (레벨 1~)

### 마그넷 바 (Magnet Bar)
- **ID:** `magnetBar` | **중첩:** 최대 3회
- **효과:** 바 표시 너비 및 물리 충돌 폭 ×1.2

### 경험치 증폭 (EXP Boost)
- **ID:** `expBoost` | **중첩:** 최대 3회
- **효과:** EXP 획득량 +30% (3중첩 시 ×1.9)

### 큰 볼 (Big Ball)
- **ID:** `bigBall` | **중첩:** 최대 2회
- **효과:** 볼 반지름 ×1.3 (히트박스 포함)

### 빠른 소환 (Quick Summon)
- **ID:** `quickSummon` | **중첩:** 최대 3회
- **효과:** +볼 쿨타임 -3초 (3중첩 시 3초)

### 오버드라이브 (Overdrive)
- **ID:** `overdrive` | **중첩:** 최대 3회
- **효과:** 볼 속도 배율 ×1.2 (상한 ×2.0)

---

## ★★ Uncommon (레벨 3~)

### 고스트 볼 (Ghost Ball)
- **ID:** `ghostBall` | **중첩:** 불가
- **효과:** 볼이 브릭 1개를 관통 후 정상 튕김. 패들 반사 시 충전.

### 헤비 메탈 (Heavy Metal)
- **ID:** `heavyMetal` | **중첩:** 가능
- **효과:** ballDamage +1, 볼 속도 ×0.85

### 쉴드 (Shield)
- **ID:** `shield` | **중첩:** 최대 3회
- **효과:** N초마다 쉴드 1회 자동 충전. 볼 하단 이탈 시 자동 반사.
- **충전 주기:** 1중첩 20초 → 2중첩 13초 → 3중첩 8초
- **이펙트:** 반사 시 청록 파티클 섬광

### 분열 볼 (Split Shot)
- **ID:** `splitShot` | **중첩:** 불가
- **효과:** 브릭 파괴 시 15% 확률로 파괴 위치에서 볼 1개 추가 생성 (볼 상한 이하 시)
- **이펙트:** 노란 별 파티클

### 모멘텀 (Momentum)
- **ID:** `momentum` | **중첩:** 불가
- **효과:** 볼이 브릭을 연속 타격할수록 데미지 +1 스택 (최대 +3). 패들 반사 시 리셋.
- **이펙트:** 스택마다 볼 색상 흰→주황→빨 변화

---

## ★★★ Rare (레벨 5~)

### 전격 볼 (Electric Ball)
- **ID:** `electricBall` | **중첩:** 불가
- **효과:** 볼이 브릭 명중 시 같은 열(column)의 랜덤 브릭 1개에 1 추가 피해
- **이펙트:** 번개 아크 라인 + 노란 스파크

### 폭발탄 (Bomb Shot)
- **ID:** `bombShot` | **중첩:** 불가
- **효과:** 브릭 파괴 시 20% 확률로 주변 3×3 범위 전체에 ballDamage 피해
- **이펙트:** 오렌지 폭발 파티클 다량

### 노바 버스트 (Nova Burst)
- **ID:** `novaBurst` | **중첩:** 불가
- **효과:** 패들 반사 시마다 반경 200px 내 모든 브릭에 1 피해
- **이펙트:** 방사형 청록 충격파 링

### 인페르노 (Inferno)
- **ID:** `inferno` | **중첩:** 불가
- **효과:** 브릭 파괴 시 같은 행 전체로 화염 전파 → 행 내 모든 브릭에 1 피해
- **이펙트:** 불꽃이 좌우로 퍼져나가는 파티클

---

## ★★★★ Legendary (레벨 8~)

### 히드라 바 (Hydra Bar)
- **ID:** `hydraBar` | **중첩:** 불가
- **효과:** 패들 반사 시 ±30° 방향으로 볼 2개 추가 생성 (총 3분열). 상한: HYDRA_MAX_BALLS + extraBalls
- **이펙트:** 분열 섬광

### 타임 스탑 (Time Stop)
- **ID:** `timeStop` | **중첩:** 불가
- **효과:** 패들 반사 시마다 모든 브릭 0.8초 동결. 볼이 많을수록 사실상 브릭이 멈춤.
- **이펙트:** 청색 동결 파동이 화면 전체로 확장

---

## 선택 규칙

| 항목 | 내용 |
|---|---|
| 표시 개수 | 3개 (가중치 기반 비복원 샘플링) |
| Common 등장 조건 | 레벨 1~ |
| Uncommon 등장 조건 | 레벨 3~ |
| Rare 등장 조건 | 레벨 5~ |
| Legendary 등장 조건 | 레벨 8~ |
| 등급 가중치 | Common 50 / Uncommon 35 / Rare 15 / Legendary 5 |

## 주요 시너지

- `inferno` + `bombShot` → 행 화염 전파 중 추가 폭발 발동 가능
- `novaBurst` + `hydraBar` → 볼 8개 × 매 반사 충격파 → 화면 폭발 도배
- `timeStop` + `hydraBar` → 볼 많을수록 초당 수십 번 동결 → 브릭 사실상 정지
- `splitShot` + `hydraBar` → 분열볼도 패들에 닿으면 3분열
- `momentum` + `heavyMetal` → 고데미지 + 콤보 스택으로 후반 고HP 브릭 순삭
