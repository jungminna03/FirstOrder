개인정보처리방침

앱명: First Order: Brick Chaos
최종 업데이트: 2026년 3월

1. 수집하는 정보

본 앱은 어떠한 개인정보도 수집하지 않습니다. 계정 등록, 이름, 이메일, 위치 정보, 기기 식별자 등을 수집하거나 저장하지 않습니다.

2. 로컬 저장 데이터

게임 진행 상황(최고 점수, 획득한 샤드, 업그레이드 레벨, 사운드 설정)은 오직 사용자 기기의 로컬 저장소(localStorage)에만 저장됩니다. 이 데이터는 외부 서버로 전송되지 않습니다.

3. 제3자 서비스

본 앱은 광고 네트워크, 분석 도구, 소셜 미디어 SDK 등 어떠한 제3자 서비스도 사용하지 않습니다.

4. 인터넷 사용

앱은 인터넷 권한을 보유하고 있으나, 이는 WebView 기반 실행 환경에서 필요한 기술적 요건입니다. 게임 플레이는 오프라인에서도 완전히 작동합니다.

5. 아동 개인정보 보호

본 앱은 어떠한 연령의 사용자로부터도 개인정보를 수집하지 않습니다.
